🚀 A sleek and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.
Test your knowledge, challenge your brain, and enjoy a smooth & engaging quiz experience.

## ✨ Features

✔️ 📝 Multiple Choice Questions (MCQs)
✔️ 🎯 Instant Answer Selection
✔️ 📊 Real-time Score Tracking
✔️ 🔄 Restart Quiz Anytime

## 🧠 How It Works

1️⃣ Start the quiz
2️⃣ Choose the correct answer
3️⃣ Move through questions
